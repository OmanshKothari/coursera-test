Internal Linking

It involves linking the pages that are the part of same web app.
  1. Relative Linking :
    This type of linking means that the link is to a variabe reference [No fixed URL] which will change over time.
  2. Absolute linking :
    This type of linking means that the link is to a fixed reference [Fixed URL] which will not change over time.
